About Me

As a recent CS graduate and freshly minted full stack software engineer I am constantly learning new technologies and improving my skills. Below is a portfolio of some of the projects I have worked on and tech stacks I am interested in.

ABC Bank

Tata Consultancy Services

ABC Bank is a fully functional banking application done as a group project as part of my training at TCS. Users can sign in and create an account and make deposits and withdrawls. Bank administrators can view current users and add and delete them. The backend was written in Java Server Pages and the database was Oracle SQL Database.

HealthTrack

Wayne State University

HealthTrack is a cross-platform health tracking and storage mobile application compatible with IOS and Android developed using the Flutter/Dart SDK. Users can input daily vitals, medications, and doctor's appointments and track usage. Intuitive push notifications are generated to prompt users to track data and take medications.

Contact Flow

RM Homecare

Executed a decision-making model upon collecting contact information, new contacts are created, and existing contacts are updated. Customized directly for Salesforce.com platform to manage large contact lists.

Python House

Wayne State University

Created a desktop application for users to input the size of the house and how many sides, windows, and doors to illustrate, the shapes of the components are inherited from the parent class polygon.Implemented the Python turtle library to illustrate the graphics.
